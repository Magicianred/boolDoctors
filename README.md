# BoolDoctors
### Progetto di fine corso
BoolDoctors Ã¨ una web app responsive che permette agli utenti di ricercare medici specialisti e di interagire con loro inviando messaggi o dando recensioni. I medici possono iscriversi e creare un proprio profilo personale in cui hanno modo, oltre di indicare i propri dati, di aggiungere le prestazioni che eseguono e acquistare un piano di sponsorizzazione per risultare tra i medici in evidenza.

Le tecnologie utilizzate sono:
* PHP con Laravel
* Braintree
* Javascript con VueJS
* chiamate Ajax
* SCSS
\
&nbsp;
\

I comandi da lanciare per visualizzare il progetto sono i seguenti:
* composer install
* npm install
* copy .env.example .env 
* importare il database e modificare il file .env
* php artisan key:generate
* php artisan storage:link
